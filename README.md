# Chicken Wings Telegram Order Bot

🚀 Telegram bot for placing and managing fast-food orders.

### 🔹 Features

✅ Firebase-based product management  
✅ Google Sheets for order history logging  
✅ Multilingual support (Uzbek, Russian, English)  
✅ Integrated payment methods (Click, Payme, Cash)

### 🔧 Tech Stack

- Python, Flask
- Firebase, Google Sheets API
- Telegram Bot API

### 📖 Installation

1. Clone the repository:
   ```bash
   git clone <repository_url>
   cd <repository_name>
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Configure environment variables:
    - Create a `.env` file in the root directory.
    - Add the following required values:
      ```
      TELEGRAM_BOT_TOKEN=<your_telegram_bot_token>
      FIREBASE_CREDENTIALS=<your_firebase_credentials_file>
      SHEETS_API_KEY=<your_google_sheets_api_key>
      ```

4. Start the bot:
   ```bash
   python main.py
   ```

### 🚀 Usage

1. **Interact with the bot:** Send `/start` to initiate the conversation with the bot.
2. **Place an Order:**
    - Select products from the list.
    - Choose the desired quantity.
    - Confirm the order and choose a payment option (Click, Payme, or Cash).
3. **Order Logging:** Orders will be logged in a connected Google Sheet for tracking.
